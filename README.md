

# Player Movement in Pygame

A simple Pygame project demonstrating basic player movement using the arrow keys or WASD keys.

## Features
- Move the player character in four directions (up, down, left, right).
- Smooth movement using Pygame's event loop.
- Easy to extend for adding enemies, obstacles, or game goals.

## Requirements
- Python 3.7+
- Pygame (`pip install pygame`)

## Installation
1. Clone the repository:
```git clone https://github.com/Pujan-Dev/Player-Movement-in-pygame```

2. Navigate to the project folder:

   ```bash
   cd player-movement-pygame
   ```
3. Install Pygame:

   ```bash
   pip install pygame
   ```

## How to Run
```
python main.py
```
* Use **Arrow Keys** or **WASD** to move the player around the screen.
